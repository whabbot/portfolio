# Project Detail Typography Alignment Design

## Goal

Bring the `ProjectDetail` page typography and spacing closer to the quieter scale used on the projects grid.

## Approved direction

Use the project grid description as the reference point for the detail-page summary and related body copy rhythm.

- Keep the project-detail page title larger than the project-tile title.
- Reduce the summary text under the title so it sits much closer to the grid description scale.
- Check section body copy and nearby vertical spacing so the header and case-study sections feel part of the same system.
- Keep the page readable, but avoid the oversized intro feel from the previous version.

## Notes

- This is a styling/layout-only pass.
- No data model, route, or behavior changes are required.
